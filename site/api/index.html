API Documentation

Programmatic access to ChinaXiv Translations data through our REST API.

Getting Started

Our API provides programmatic access to our translation database. All endpoints return JSON data and support CORS for web applications.

Base URL

https://chinaxiv-english.pages.dev/api/

Rate Limits

Currently no rate limits are enforced, but please be respectful with your usage. If you need higher limits, please contact us.

Endpoints

GET /api/papers

Retrieve a list of all papers with optional filtering and pagination.

Parameters

Parameter Type Description Example
limit integer Number of results to return (max 100) ?limit=20
offset integer Number of results to skip ?offset=40
category string Filter by category ?category=cs
date_from date Filter papers from this date (YYYY-MM-DD) ?date_from=2025-01-01
date_to date Filter papers to this date (YYYY-MM-DD) ?date_to=2025-12-31

Example Request

GET /api/papers?limit=10&category=cs&date_from=2025-01-01

Example Response

{
  "papers": [
    {
      "id": "chinaxiv-202510.00001",
      "title": "Machine Learning Applications in Computer Vision",
      "authors": "Zhang Wei, Li Ming",
      "abstract": "This paper presents...",
      "subjects": "Computer Science, Machine Learning",
      "date": "2025-10-01",
      "url": "https://chinaxiv-english.pages.dev/items/chinaxiv-202510.00001/"
    }
  ],
  "total": 150,
  "limit": 10,
  "offset": 0
}

GET /api/papers/{id}

Retrieve detailed information about a specific paper.

Parameters

Parameter Type Description
id string Paper ID (e.g., chinaxiv-202510.00001)

Example Request

GET /api/papers/chinaxiv-202510.00001

Example Response

{
  "id": "chinaxiv-202510.00001",
  "title": "Machine Learning Applications in Computer Vision",
  "authors": "Zhang Wei, Li Ming",
  "abstract": "This paper presents novel approaches...",
  "subjects": "Computer Science, Machine Learning",
  "date": "2025-10-01",
  "url": "https://chinaxiv-english.pages.dev/items/chinaxiv-202510.00001/",
  "original_url": "https://www.chinaxiv.org/abs/202510.00001",
  "markdown_url": "https://chinaxiv-english.pages.dev/items/chinaxiv-202510.00001/chinaxiv-202510.00001.md"
}

GET /api/search

Search papers by query string with optional filters.

Parameters

Parameter Type Description Example
q string Search query ?q=machine learning
limit integer Number of results to return (max 100) ?limit=20
category string Filter by category ?category=cs

Example Request

GET /api/search?q=neural networks&limit=5

Example Response

{
  "query": "neural networks",
  "papers": [
    {
      "id": "chinaxiv-202510.00001",
      "title": "Deep Neural Networks for Image Recognition",
      "authors": "Wang Lei, Chen Xiaoli",
      "abstract": "This paper explores...",
      "subjects": "Computer Science, Machine Learning",
      "date": "2025-10-01",
      "url": "https://chinaxiv-english.pages.dev/items/chinaxiv-202510.00001/"
    }
  ],
  "total": 25,
  "limit": 5
}

Error Handling

All API endpoints return appropriate HTTP status codes and error messages in JSON format.

Common Error Responses

404 Not Found

{
  "error": "Paper not found",
  "message": "The requested paper ID does not exist",
  "status": 404
}

400 Bad Request

{
  "error": "Invalid parameters",
  "message": "Limit must be between 1 and 100",
  "status": 400
}

500 Internal Server Error

{
  "error": "Internal server error",
  "message": "An unexpected error occurred",
  "status": 500
}

Usage Examples

JavaScript/Fetch

// Get recent papers
fetch('https://chinaxiv-english.pages.dev/api/papers?limit=10')
  .then(response => response.json())
  .then(data => {
    console.log('Found', data.total, 'papers');
    data.papers.forEach(paper => {
      console.log(paper.title);
    });
  })
  .catch(error => {
    console.error('Error:', error);
  });

// Search for papers
fetch('https://chinaxiv-english.pages.dev/api/search?q=machine learning&limit=5')
  .then(response => response.json())
  .then(data => {
    console.log('Search results:', data.papers);
  });

Python/Requests

import requests

# Get papers by category
response = requests.get('https://chinaxiv-english.pages.dev/api/papers', 
                       params={'category': 'cs', 'limit': 20})
data = response.json()

print(f"Found {data['total']} papers")
for paper in data['papers']:
    print(f"- {paper['title']}")

# Search for papers
search_response = requests.get('https://chinaxiv-english.pages.dev/api/search',
                              params={'q': 'neural networks', 'limit': 10})
search_data = search_response.json()

print(f"Search results: {len(search_data['papers'])} papers found")

cURL

# Get all papers
curl "https://chinaxiv-english.pages.dev/api/papers"

# Search for papers
curl "https://chinaxiv-english.pages.dev/api/search?q=machine%20learning&limit=10"

# Get specific paper
curl "https://chinaxiv-english.pages.dev/api/papers/chinaxiv-202510.00001"

# Get statistics
curl "https://chinaxiv-english.pages.dev/api/stats"

Data Formats

Paper Object

Field Type Description
id string Unique paper identifier
title string Paper title (translated)
authors string Comma-separated list of authors
abstract string Paper abstract (translated)
subjects string Comma-separated list of subjects/categories
date string Publication date (YYYY-MM-DD)
url string URL to the translated paper page
original_url string URL to the original Chinese paper
markdown_url string URL to the markdown version
