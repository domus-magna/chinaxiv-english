Help & FAQ

Find answers to common questions about using ChinaXiv Translations and learn about our translation process.

Getting Started

What is ChinaXiv Translations?

ChinaXiv Translations is a project that automatically translates academic papers from ChinaXiv (the Chinese equivalent of arXiv) into English, making Chinese research accessible to English-speaking researchers worldwide.

How do I search for papers?

You can search for papers in several ways:

  • Use the search box in the header to search across all papers
  • Visit our Advanced Search page for more filtering options
  • Browse by category using the links in the footer
  • Use the Browse All page to explore our collection

How often is new content added?

We automatically harvest and translate new papers from ChinaXiv on a daily basis. The site is updated every night with the latest papers.

About Translations

How accurate are the translations?

Our translations are generated using advanced AI language models (DeepSeek V3.2-Exp). While they are generally accurate, especially for technical content, we recommend:

  • Always verify important information with the original Chinese source
  • Check mathematical formulas and equations carefully
  • Consider the context when interpreting results

Are mathematical formulas translated correctly?

Mathematical formulas, equations, and LaTeX expressions are preserved exactly as they appear in the original papers. Only the surrounding text is translated.

Can I access the original Chinese papers?

Yes! Each translated paper includes a link to view the original on ChinaXiv. We encourage researchers to refer to the original source for the most accurate information.

Using the Site

How do I cite these papers?

We recommend citing the original Chinese paper from ChinaXiv rather than our translation. Each paper page includes citation information and BibTeX export options.

Can I download papers?

Currently, papers are available to read online. We're working on adding PDF download functionality. You can always access the original papers on ChinaXiv for download.

Is there an API available?

Yes! We provide an API for programmatic access to our translation database. Visit our API documentation for details.

How do I bookmark papers?

You can bookmark papers using the bookmark button on individual paper pages. Bookmarks are stored locally in your browser.

Technical Issues

Why don't I see any search results?

If you're not seeing search results, try:

  • Using broader search terms
  • Checking your internet connection
  • Refreshing the page
  • Clearing your browser cache

Are there any browser requirements?

Our site works with all modern browsers. For the best experience, we recommend:

  • Chrome, Firefox, Safari, or Edge (latest versions)
  • JavaScript enabled
  • Stable internet connection

Why are some pages loading slowly?

Large search results or complex mathematical content may take longer to load. If pages are consistently slow, please let us know through our contact page.

Support & Contact

How can I report a problem?

If you encounter any issues or have suggestions for improvement, please contact us through our contact page or open an issue on our GitHub repository.

How can I support this project?

We appreciate any support! You can:

  • Make a donation to help cover API and server costs
  • Share our site with colleagues and researchers
  • Report bugs or suggest improvements
  • Contribute to our open-source project on GitHub

Can I request specific papers to be translated?

Currently, we automatically translate all papers from ChinaXiv. If you need a specific paper translated urgently, please contact us and we'll see what we can do.

Still have questions?

If you can't find the answer you're looking for, please don't hesitate to contact us. We're here to help!
